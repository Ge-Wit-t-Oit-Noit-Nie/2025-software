\hypertarget{md_docs_2coding__conventions_autotoc_md6}{}\doxysubsection{\texorpdfstring{lower\+\_\+case}{lower\+\_\+case}}\label{md_docs_2coding__conventions_autotoc_md6}
Wanneer je eigen bestanden toevoegd (zoals in de {\ttfamily docs/} folder), dan wordt dit altijd in {\itshape lower\+\_\+case} gedaan.

\begin{DoxyNote}{Note}
Als uitzondering zijn "{}\+CONTRIBUTING.\+md"{}, "{}\+README.\+md"{} en "{}\+LICENSE"{}. Deze volgen een algemene conventie.
\end{DoxyNote}
\begin{DoxyImportant}{Important}
Bestanden die gemaakt zijn STM32\+Cube\+MX zijn ook uitgezonderd.
\end{DoxyImportant}
